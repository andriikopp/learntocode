Змінні та оператори
Дано три числа a, b та c,

знайдіть їх суму
a + b + c

знайдіть їх добуток
a * b * c

знайдіть їх середнє значення
(a + b + c) / 3

знайдіть квадрати цих чисел
a * a, b * b, c * c

знайдіть квадратні корені цих чисел
sqrt(a), sqrt(b), sqrt(c)

знайдіть модулі різниць усіх пар чисел
abs(a – b), abs(a – c),
abs(b – c)

знайдіть неповні частки та остачи від ділення чисел на 2
a / 2 and a % 2, ...

Прийняття рішень
Дано два числа a та b,

знайдіть найбільше число
if a > b then print a else b

знайдіть найменше число
if a < b then print a else b

знайдіть їх суму, якщо числа рівні, або їх добуток
if a = b then a + b else a * b

знайдіть їх квадратні корені, якщо числа є додатними, або модулі тільки від'ємних чисел
if a > 0 and b > 0 then sqrt(a), sqrt(b)
else
if a < 0 then abs(a), if b < 0 then abs(b)

Прийняття рішень
Дано номер місяця n,

визначте пору року, до якого він належить
if n < 0 or n > 12 then warning
else if n = 12 or n < 3 then winter
else if n < 6 then spring
else if n < 9 then summer
else autumn

Прийняття рішень
Дано три числа a, b та c,

знайдіть найбільшу суму пар цих чисел
if a + b > a + c and a + b > b + c then a + b ...

знайдіть число, якому дорівнює сума двох інших чисел, або добуток усіх чисел
if a + b = c then print c
else if a + c = b then print b
else if b + c = a then print a
else print a * b * c

знайдіть число, яке більше за суму двох інших чисел, або суму усіх чисел
if a + b < c then print c
else if a + c < b then print b
else if b + c < a then print a
else print a + b + c
